# Jetpack.js

This is the website front-end to communicate with a Jetpack ESP32 device.

# Development

Before you can build the project, make sure you:
- Installed NodeJS
- Run `npm install`

Continuous building with local web server:

```bash
npm run start
```

Building static files into `dist/`:

```bash
npm run build
```
